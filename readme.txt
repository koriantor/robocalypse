SETUP:
	(1) From cmd line, go to js folder.  Install:
			Twitter (npm install twitter)
			Browserify (npm install browserify)

	(2) Browserify package
			"browserify app.js > bundle.js"
	
	(3) Load Extension into chrome in developer mode.
